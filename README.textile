h3. Install GD

<pre>
	<code>
		mkdir ~/src
		cd ~/src
		curl -O http://www.libgd.org/releases/gd-latest.tar.gz
		tar -xfz gd-latest.tar.gz
		cd gd-2.0.34
		./configure
		make
		sudo make install
	</code>
</pre>
